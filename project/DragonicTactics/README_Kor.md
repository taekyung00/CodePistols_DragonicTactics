# 드래고닉 택틱스 (Dragonic Tactics)

## 개요 (Summary)

- **게임 이름**: 드래고닉 택틱스 (Dragonic Tactics)

- **버전**: 0.1.3524

- **개발자**: 기준영, 송승주, 이상윤, 허태경, 박기남

- **팀 이름**: 코드 피스톨즈 (Code Pistols)

- **게임 설명**: 이 게임은 전통적인 D&D 캠페인의 전투를 재해석한 작품으로, 플레이어가 드래곤이 되어 모험가 파티에 맞서 싸우는 게임입니다. 게임 내의 각 캐릭터는 자신의 턴을 가집니다. 플레이어는 '드래곤'이라 불리는 캐릭터가 되어 자신의 집으로 침입하는 모험가들을 쫓아내야 합니다. 버튼을 클릭하여 행동, 이동, 공격을 결정할 수 있으며, 마우스 클릭만으로 모든 조작이 가능합니다.

## CMake 프리셋 (CMake Presets)

CMake 프리셋은 복잡한 설정들을 미리 구성하여 빌드 과정을 단순화합니다. 다음과 같은 항목들을 수동으로 지정하는 대신 자동으로 처리합니다.

- 컴파일러 플래그 및 최적화 레벨

- 출력 디렉토리 및 빌드 경로

- 플랫폼별 구성

- 종속성 위치

- 디버그/릴리스 전용 설정

예를 들어, `cmake --preset linux-debug`를 실행하는 것은 수많은 플래그를 포함한 복잡한 명령어를 실행하는 것과 동일한 효과를 냅니다. 이는 시간을 절약하고 여러 플랫폼 및 빌드 유형에서 작업할 때 발생할 수 있는 구성 오류를 줄여줍니다.

### 게임 실행 및 조작 (How to play)

- 프로젝트 빌드 후, 빌드 디렉토리에 생성된 실행 파일을 실행하세요.

- 예: `windows-debug` 프리셋으로 빌드했다면, `./build/windows-debug/dragonic_tactics.exe`를 실행합니다.

- 메인 메뉴에서 "Dragonic Tactics"를 클릭하여 게임을 시작합니다.

- 게임이 시작되면 각 캐릭터의 속도(Speed) 능력치에 의해 턴 순서가 결정됩니다.

- 전투 상태(Combat Status) 창에서 턴 순서를 확인할 수 있습니다.

- 플레이어는 이동(Move), 행동(Action), 턴 종료(End Turn) 중 하나를 선택할 수 있습니다.

- **이동 (Move)**: 이동하고 싶은 그리드를 클릭하세요. 각 플레이어는 자신의 속도 능력치만큼 이동할 수 있습니다. 예를 들어 속도가 4라면 최대 4칸까지 이동 가능합니다. 속도보다 적게 이동하거나, 2칸 이동 후 행동을 하고 다시 2칸을 이동하는 등 이동과 행동을 섞어서 사용할 수 있습니다. 캐릭터의 속도는 오른쪽 UI에 표시됩니다.

- **행동 - 기본 공격 (Action - basic attack)**: "Action" 버튼을 누른 후 "Attack"을 클릭하여 기본 공격을 사용합니다. 현재는 기본 공격만 구현되어 있습니다. (주문 버튼을 클릭한 경우 오른쪽 마우스 버튼을 눌러 취소할 수 있습니다.) 적이 드래곤(빨간색 캐릭터)의 사거리(현재 2칸) 내에 있다면 적을 클릭하여 공격하세요. 피해량은 공격자의 주사위 기반 공격력에서 방어자의 주사위 기반 방어력을 뺀 값으로 계산됩니다. 사거리 내에 적이 없으면 아무 일도 일어나지 않습니다. 드래곤은 자신의 공격 포인트(AP)에 따라 공격할 수 있으며, AP는 오른쪽 UI에 표시됩니다. 기본 공격 한 번당 1 AP를 소모합니다.

- **턴 종료 (End Turn)**: "End Turn" 버튼을 클릭하여 턴을 마칩니다. 이동 거리나 AP가 남아있어도 언제든지 턴을 종료할 수 있습니다.

- 캐릭터 중 한 명의 HP가 0이 되면 게임이 종료됩니다.
