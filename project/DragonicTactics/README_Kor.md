# 드래고닉 택틱스 (Dragonic Tactics)

## 🎮 게임 실행 방법

- 게임 폴더 내의 `.exe` 파일을 실행하여 플레이할 수 있습니다.
- **메인 화면 조작:** 화면에서 '드래고닉 택틱스'를 마우스 왼쪽 버튼으로 클릭하거나, 해당 글자가 초록색으로 활성화되었을 때 `Enter` 키를 입력하면 인게임으로 진입합니다.

## 📜 게임 규칙 및 승리 조건

- 플레이어와 적 인공지능(AI)이 번갈아 가며 진행하는 턴제 전략 게임입니다.
- 턴이 종료되면 적들의 턴이 시작되며, AI가 플레이어를 공격해옵니다.
- **승리 조건:** 맵에 존재하는 모든 적을 쓰러뜨리면 승리합니다.

## ⌨️ 단축키 및 조작 요약

- **`마우스 좌클릭`**: 캐릭터/타일 선택, 이동, 스킬 사용
- **`마우스 우클릭`**: 스킬 범위 표시 취소
- **`Enter`**: 인게임 진입 (메인 화면의 타이틀 글자가 초록색일 때)
- **`ESC`**: 인게임에서 메인 화면으로 복귀

## 🕹️ 시스템 가이드

- **상태 확인:** 드래곤이나 다른 캐릭터 위에 마우스 커서를 올리면 체력(HP), 행동력(AP), 이동력(Speed), 스펠 슬롯을 확인할 수 있습니다.
- **이동 범위:** 드래곤의 턴에 커서를 올리면 이번 턴에 이동할 수 있는 타일의 범위가 표시됩니다.

### 💧 자원 관리

- **Speed (이동력):** 드래곤을 클릭한 후 초록색 타일을 누르면 이동하며, 거리에 따라 Speed 자원을 소모합니다.

- **AP (행동력):** 공격이나 스킬을 사용하면 AP 자원을 소모합니다.

- **스펠 슬롯:** 마법 스킬 사용 시 AP와 함께 소모됩니다.
  
  > **※ 주의:** Speed와 AP는 매 턴 시작 시 완전히 복구되지만, **소모된 스펠 슬롯은 자동으로 회복되지 않습니다.**

### ✨ 업캐스팅 (Upcasting)

일반공격을 제외한 마법 스킬(파이어볼트 ~ 벽생성) 중 일부는 **더 높은 레벨의 스펠 슬롯**을 소모하여 스킬의 위력을 강화할 수 있습니다. 한정된 스펠 슬롯을 확인하고 상황에 맞게 마법을 강화해 보세요.

## ⚔️ 스킬 목록

화면 하단의 아이콘(좌측부터 순서대로)을 클릭하여 스킬을 사용할 수 있습니다.

- **일반공격**
- **파이어볼트 (1레벨)**: 적 단일 대상 / 사거리 4칸 / 업캐스팅 가능
  - 피해량: `2d8 + (소모 스펠레벨 - 1)d6`
- **테일스위프 (2레벨)**: 시전자 주변 모든 적 / 사거리 2칸 / 업캐스팅 불가
  - 피해량: `1d8`
  - 특수 효과: 피격된 적은 드래곤을 중심으로 2칸 멀리 밀려남
- **용의분노 (3레벨)**: 직선 범위 내 모든 적 / 사거리 4칸 / 업캐스팅 가능
  - 피해량: `4d6 + (소모 스펠레벨 - 3) * 2d6`
- **메테오 (3레벨)**: 맵 전체 모든 적 / 업캐스팅 가능
  - 피해량: `3d20 + (소모 스펠레벨 - 3) * 1d20`
  - 특수 효과: 시전자에게 1턴간 '탈진(Exhaustion)' 상태 부여
- **마력전환 (0레벨)**: 자신 / 사거리 0칸 / 업캐스팅 가능
  - 효과: 자신에게 `(소모 스펠레벨 + 1)d10` 만큼의 피해를 입히고, `(소모 스펠레벨 + 1)` 레벨의 스펠 슬롯을 1개 회복
- **정화 (1레벨)**: 자신 / 사거리 0칸 / 업캐스팅 불가
  - 효과: 1턴간 자신에게 '정화(Purify)' 상태 부여
- **공포의 외침 (1레벨)**: 시전자 주변 모든 적 / 사거리 3칸 / 업캐스팅 불가
  - 효과: 3턴간 범위 내 모든 적들에게 "공포(Fear)"상태 부여
- **용암발사 (2레벨)**: 빈 타일 지정 / 사거리 6칸 / 업캐스팅 가능
  - 효과: 선택한 위치에 용암 지대 소환. `(소모 스펠레벨 - 2)` 만큼 타일 추가 생성
- **벽생성 (1레벨)**: 빈 타일 지정 / 사거리 5칸 / 업캐스팅 가능
  - 효과: 선택한 위치에 벽 소환. `(소모 스펠레벨 - 1)` 만큼 타일 추가 생성
- **턴 종료**

## 💫 상태이상 (Status Effects)

캐릭터의 스킬이나 특정한 조건에 의해 부여되는 버프 및 디버프 효과입니다.

- **흡혈 (Lifesteal):** 이번 턴에 주는 모든 피해의 절반만큼 체력을 회복합니다. (소수점 버림)
- **광란 (Frenzy):** 다음 공격이 10 이상의 피해를 입히면 피격된 적이 광란을 제외한 무작위 해로운 상태이상(저주, 공포, 탈진 중 1)을 받습니다. 실패할 경우 시전자(파이터)가 해당 무작위 해로운 상태이상을 받습니다.
- **탈진 (Exhaustion):** 다음 턴에 Speed와 AP(행동력)가 0이 됩니다.
- **정화 (Purify):** 자신에게 걸린 모든 상태이상을 즉시 제거합니다.
- **축복 (Blessing):** 받는 모든 피해가 3 감소하고, 주는 모든 피해가 3 증가합니다.
- **저주 (Curse):** 받는 모든 피해가 3 증가하고, 주는 모든 피해가 3 감소합니다.
- **신속 (Haste):** Speed가 1 증가하고, AP(행동력)가 1 증가합니다.
- **은신 (Stealth):** 적의 타겟이 되지 않으며, 다음 턴에 주는 첫 피해가 2배로 적용됩니다. 공격 후에는 사용할 수 없으며, 공격 시 즉시 은신이 해제됩니다.
- **공포 (Fear):** 주는 모든 피해가 3 감소하고, Speed가 1 감소합니다.

## ⚠️ 알려진 버그 (Known Issues)

- 인게임 화면에서 `ESC` 키를 눌러 메인 화면으로 돌아갈 경우, 인게임 배경음악(BGM)이 중복으로 계속 재생되는 현상(수정됨)
- 스킬 아이콘을 눌러 범위가 표시된 상태에서 `우클릭`으로 취소하지 않고 일반공격을 사용할 경우, 화면에 이전 스킬의 범위가 계속 남아있는 현상(수정됨)
- 캐릭터의 체력이 0이 되어도 화면에서 캐릭터 스프라이트가 사라지지 않는 현상(수정됨)
- 일반공격의 사거리(범위)가 시각적으로 표시되지 않는 현상(수정됨)
- 메테오 사용 시, 플레이어(시전자)가 아닌 적들이 기절 상태에 빠지는 현상(수정됨)
