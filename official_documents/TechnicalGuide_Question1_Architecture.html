Architecture Overview

Dragonic Tactics is built on a custom C++20 game engine with a modular, component-based architecture. The engine separates core systems (Engine layer), rendering abstraction (CS200 layer), low-level graphics (OpenGL layer), and game-specific logic (DragonicTactics layer).

This architecture follows key principles: modular independence, event-driven communication, data-driven design, and component-based composition.
