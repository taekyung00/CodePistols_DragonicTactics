Architecture Overview

Dragonic Tactics is built on a custom C++20 game engine with a modular, component-based architecture.

The engine separates systems into four distinct layers:

Game Logic Layer

DragonicTactics namespace - game-specific code

Engine Core Layer

CS230 namespace - reusable engine systems

Rendering Abstraction

CS200 namespace - graphics API abstraction

OpenGL Wrapper

Low-level OpenGL 3.3+ wrapper
